Setup

Step 1

Create a Google Cloud Console account.

Create a new project, for example BobBot.

Step 2

Go to Dialogflow Console

Create a new agent and choose the ProjectID created from the Google Cloud Console.

Step 3

Go to Intents and update Default Welcome Intent.

Step 4

Create a new intent - Hours to display the working hours of the bike shop..

Step 5

Create a new intent to take appointments - Make Appointment.

Step 6

Set the action parameters to understand date and time. Since we are booking an appointment,
it is required to have a date and time. Select the required option for data and time parameters
and update the prompts. Example: What day works for you?.

Step 7

Create responses for the user queries.

Step 8

Create a follow-up intent for the Make Appointment intent. Here, after booking an appointment,
we would like to ask the type of service the user needs, i.e a repair or a maintenance,
to make the bot more realistic.

Step 9

Add the follow-up intent.

Step 10

Update the follow-up intent by providing training phases and responses and choose the Entity.

The response should look something like this: “Okay, we'll schedule a $appointmenttype(entity value) on
#MakeAppointment-followup.date(parameter from Make Appointment Intent)
at #MakeAppointment-followup.time(parameter from Make Appointment Intent). We'll see you then.”

Step 11

Go to Integrations and choose WebDemo. It opens a demo application embedded with the BobBot.
