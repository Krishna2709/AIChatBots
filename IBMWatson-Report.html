Total Time: 7hrs | Platform: IBM Cloud

Demo: BobBot

Setup

Step 1

Create a IBM Cloud account.

Go to Resource groups and create a new one.

Step 2

Go to Cloud Foundry Orgs and create a new Org and add a Space with the Resource group id.

Step 3

Go to Catalog and select IBM Cloud CLI and install the IBM Cloud CLI tool into your local machine. Check the installation by ibmcloud help.

Step 4

Go to Catalog and select Watson Assistant.

Step 5

Create a new Watson Assistant by providing a name and save the credentials or the API keys.

Step 6

Download the git repository to your local machine. Simple Watson Assistant

Step 7

Go to the Watson Assistant, launch the assistant created earlier and visit the Dialog section from the menu.
Upload the bank_simple_workspace.json from the training folder in the downloaded git repository.

Step 8

Open a command prompt in the root directory of the git repository.

Step 9

Copy contents from .env.examples to .env and update the credentials.

> copy .env examples .env

Step 10

Update the values in the .env file from the API credential keys downloaded earlier.

Step 11

Go to manifest.yml file and change the app name. Ex: name: watson-bob-bot.

Step 12

Install dependencies: Download Node.js and install NPM.

> npm install

Step 13

Check the installation by running the demo application. Visit the localhost

> npm start

Step 14

Login into the IBM Cloud using the CLI.

> ibmcloud login

Step 15

Install cloud foundry tools.

> ibmcloud cf install

Step 16

Specify the target Org, Resource and Space(created from the steps 2 and 3).

    > ibmcloud target -o Org

    > ibmcloud target -g Resource

    > ibmcloud target -s Space

    > ibmcloud target - -c

Step 17

Push the app into IBM Cloud

> ibmcloud app push

Step 18

Visit the generated web address: watson-bob-bot.us-east.mybluemix.net
